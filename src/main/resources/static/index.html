Reporting Guide

Required information for each submission

Item title

Use the actual item name such as student ID card, wallet, calculator, phone, or keys.

Location

Mention the last known or found location as precisely as possible, such as library gate or lab room.

Description

Include visible identifying details only and avoid unnecessary text.

Contact

Provide a reliable phone number or email so the item can be returned without delay.
