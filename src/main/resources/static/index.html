Common Questions

How to use the recovery portal.

Visit the university security desk at Level 0, Knowledge Tower with your Student ID and be prepared to describe the item in detail.

Submit a "Found" report on this portal and then drop off the item at the nearest security post or the main Lost & Found office.

Your contact information is only visible to the system administrators and the person who correctly verifies ownership of the item.
